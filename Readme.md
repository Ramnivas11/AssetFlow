AssetFlow

AssetFlow is an Enterprise Asset & Resource Management System that helps organizations manage physical assets, shared resources, maintenance, audits, and allocations from a centralized platform.

Features

- 🔐 Secure authentication with role-based access control
- 🏢 Department, employee, and asset category management
- 📦 Asset registration and lifecycle tracking
- 👤 Asset allocation and transfer workflow
- 📅 Shared resource booking with conflict detection
- 🛠️ Maintenance request and approval workflow
- 📋 Asset audit cycles and discrepancy reporting
- 📊 Reports, analytics, and KPI dashboard
- 🔔 Notifications and activity logs

Tech Stack

- Backend: Node.js, Express.js, TypeScript
- Database: PostgreSQL (Neon), Prisma ORM
- Authentication: JWT, bcrypt
- Validation: Zod

Getting Started

git clone <repository-url>
cd assetflow
npm install
npm run dev
